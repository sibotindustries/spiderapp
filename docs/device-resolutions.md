## Breakpoints Recomendados para Media Queries

```css
/* Mobile Pequeno (até 320px) */
@media (max-width: 320px) {
  /* Ajustes para iPhones antigos e dispositivos muito pequenos */
}

/* Mobile (320px - 480px) */
@media (min-width: 321px) and (max-width: 480px) {
  /* Maioria dos smartphones em modo retrato */
}

/* Mobile Grande (481px - 767px) */
@media (min-width: 481px) and (max-width: 767px) {
  /* Smartphones grandes em modo retrato, ou pequenos em paisagem */
}

/* Tablets e iPads (768px - 1023px) */
@media (min-width: 768px) and (max-width: 1023px) {
  /* Tablets em modo retrato e alguns em paisagem */
}

/* Laptops e Desktops Pequenos (1024px - 1279px) */
@media (min-width: 1024px) and (max-width: 1279px) {
  /* Tablets grandes em paisagem, notebooks e desktops pequenos */
}

/* Desktops (1280px - 1535px) */
@media (min-width: 1280px) and (max-width: 1535px) {
  /* Desktops padrão e laptops maiores */
}

/* Telas Grandes (1536px e acima) */
@media (min-width: 1536px) {
  /* Monitores grandes e telas de alta resolução */
}
```

## Orientação do Dispositivo

```css
/* Orientação Retrato */
@media (orientation: portrait) {
  /* Estilos para dispositivos em modo retrato */

/* Orientação Paisagem */
@media (orientation: landscape) {

```

## Pixel Ratio (Densidade de Pixels)

```css
/* Telas com densidade normal (1x) */
@media (-webkit-min-device-pixel-ratio: 1) {
  /* Estilos para telas padrão */
}

/* Telas Retina (2x) */
@media (-webkit-min-device-pixel-ratio: 2) {
  /* Estilos para telas de alta resolução como iPhone 8, SE */
}

/* Telas de densidade super alta (3x ou mais) */
@media (-webkit-min-device-pixel-ratio: 3) {
  /* Estilos para telas de ultra-alta resolução como iPhone 13 Pro, Galaxy S22 */
}
```

## Observações importantes para desenvolvimento responsivo

1. A maioria dos dispositivos Android modernos utiliza um tamanho de viewport de 360px CSS, independente da resolução física
2. iPhones mais recentes utilizam viewports de 375px, 390px ou 428px, dependendo do modelo
3. O pixel ratio (device-pixel-ratio) geralmente é 2.0, 3.0 ou 4.0, raramente com valores fracionados
4. Em tablets, considere breakpoints específicos para diferentes modos de uso
5. Sempre teste em dispositivos reais ou emuladores precisos quando possível
6. Algumas marcas (principalmente Huawei, Vivo e Oppo) têm proporções e tamanhos de tela menos padronizados
